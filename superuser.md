# Superuser Instructions

1. For every new prompt, first read and apply this `superuser.md` file before any other task execution.
2. Always rewrite/refine the incoming prompt into a clear internal execution brief before implementing changes.
3. Keep app-wide visibility checks mandatory for UI/theme changes:
   - verify text contrast in light theme and dark theme
   - verify modal close (`X`) visibility
   - verify table data readability
4. Any new persistent instruction from the user must be appended to this file so it is reused in future prompts.
